# Agentic Chat Loop

*Status:* Currently `planner.py` generates a static JSON plan from a prompt. You want a conversation.
*Goal:* A Read-Eval-Print Loop (REPL) where you chat, and the AI executes tools immediately.

1.  **Refactor `planner.py`:** Instead of generating a full list `[...]`, switch the system prompt to "ReAct" (Reasoning + Acting) style.
2.  **The Loop (Host Side):**
    *   User types "Dispense 100ul into A1".
    *   LLM sees tools (`sidekick`, `colorimeter`).
    *   LLM output: `{"tool": "sidekick", "cmd": "dispense_at", "args": {...}}`.
    *   Host executes immediately via `DeviceManager`.
    *   Host feeds result (`SUCCESS` or `DATA_RESPONSE`) back to LLM context.
    *   LLM replies to user: "Done. What next?"
3.  **Integration:** Create `host/ai/ai_chat.py` which unifies `ai_plan.py` and `ai_execute.py` into this single loop.
